OAUTH2 Autenticación con Google
Prerrequisitos:
1. Crear proyecto en Google Cloud Console
2. Habilitar Google+ API u OAuth 2.0
3. Crear credenciales OAuth → Web Application
4. Agregar http://localhost:8081/auth/oauth2/callback/google como Redirect URI
5. Configurar GOOGLE_CLIENT_ID y GOOGLE_CLIENT_SECRET en el backend
o simula el callback con un token
